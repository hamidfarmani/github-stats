# GitHub stats

Another way to look at your GitHub profile!

## Features

- Search GitHub profile

## Technologies

- Next.js
- Mantine
- Chart.js
- gh-polyglot

## Getting started

1. Clone the repository
   `git clone https://github.com/hamidfarmani/github-stats`

2. Install dependencies
   `yarn install`

3. Start the development server
   `yarn run dev`
